# 뉴스 대시보드 명세

## 목표

서울환경연합과 서울환경운동연합, 그리고 사업계획서에서 추출한 고유 캠페인명이 언론사 기사에서 언급되는 흐름을 저장하고 평가 근거로 정리한다.

## 기본 구조

- 화면: Vercel 정적 배포
- 저장소: Google Sheets
- 쓰기 연결: Google Apps Script 웹앱
- 정기 실행: Google Apps Script 시간 트리거, 오전 1회와 저녁 1회
- 운영 권한: 관리자와 보기 전용 두 단계

## 사용자 권한

| 권한 | 가능 작업 |
| --- | --- |
| admin | 사업계획서 업로드, 검색어 관리, 기사 검토, 품질 보정, 시트 동기화 |
| viewer | 대시보드와 기사 목록 조회 |

## 데이터 원칙

- 언론사 기사는 `source_type=media`로 저장한다.
- 보도 횟수에는 `source_type=media`, `review_status=related`인 항목이 자동 포함된다.
- 홈페이지, 기관, SNS, 블로그, 단체 자체 게시물은 이번 프로젝트에서 저장하지 않는다.
- 원문 전체를 저장하지 않고 제목, URL, 게시일, 스니펫, 판정 근거만 저장한다.
- 서로 다른 언론사의 유사 제목은 중복으로 보지 않는다.
- 같은 언론사 안에서 같은 제목이나 같은 canonical URL이 있을 때만 중복 의심으로 표시한다.

## 검색어 원칙

- 기본 조직명: 서울환경연합, 서울환경운동연합
- 사업계획서에서 고유명사 캠페인명을 추출한다.
- 예시: 플라스틱방앗간, 시티트리클럽, 씨앗의숲
- 수집은 지정 검색어 단독 검색을 먼저 실행한 뒤 사업을 매칭한다.
- 조직명+캠페인명 조합 검색은 누락 보강용으로 추가 실행한다.
- 네이버뉴스는 정확도순 정확 일치 검색을 우선 사용하고, 최신순 검색은 보조로 사용한다.
- 일반적인 `환경연합`, `환경운동연합`만 있는 지역 지부 기사는 자동 집계하지 않는다.
- 지역 지부명과 캠페인명이 함께 있는 경우 자동 집계하지 않고 검토 후보로 둔다.
- `시티트리클럽`처럼 고유성이 높은 캠페인명은 캠페인명 단독 기사도 관련 후보로 적극 분류한다.
- `플라스틱방앗간`처럼 다른 지역·기관에서도 쓰일 수 있는 캠페인명은 조직명 확인 전까지 검토 후보로 둔다.

## 사업 매칭 원칙

- 필터는 사업명 단위로 제공하며, 기사에 이미 붙어 있는 예전 사업명도 선택지에 포함한다.
- 기사 제목, 요약, 검색어에 사업명이 직접 나오면 해당 사업으로 우선 매칭한다.
- `시티트리클럽`, `플라스틱방앗간`, `지구를 구하장`처럼 고유 캠페인명이 직접 나오면 해당 캠페인 사업으로 매칭한다.
- `플라스틱`, `자원순환`, `일회용`, `재사용`, `폐기물`처럼 넓은 정책 이슈 단어만 있을 때는 행사성 캠페인보다 `정책`, `대응`, `모니터링`이 들어간 사업을 우선한다.
- 특정 사업명이 확인되지 않으면 같은 분야의 첫 번째 사업에 자동 배정하지 않고 `기타 정책 대응`을 기본값으로 둔다.

## Google Sheets 탭

| 탭 | 용도 |
| --- | --- |
| users | 사용자 권한 |
| plans | 연도별 사업계획서 스냅샷 |
| programs | 사업 단위 |
| keywords | 검색어와 캠페인명 |
| items | 언론사 기사 |
| matches | 기사와 사업의 연결 |
| fetch_runs | 수집 실행 기록 |

3. 오전/저녁 자동 수집 기록
4. 기사 후보 수집기 확장
5. 관련성, 사업 매칭, 기사 품질 자동 추천
